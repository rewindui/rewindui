# Colors
---
The `Accordion` component comes with four colors: `white`, `gray`, `zinc` and `slate`.

```tsx
type AccordionShadow = 'none' | 'sm' | 'base' | 'md' | 'lg' | 'xl';
```

# Accessibility
---
The `Accordion` component adheres to the [WAI-ARIA Accordion Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/).

## Keyboard interactions

| Key                | Description                                   |
|:-------------------|:----------------------------------------------|
| `Enter` or `Space` | Toggles the state of focused header           |
| `Tab`              | Moves focus to the next focusable element     |
| `Shift` + `Tab`    | Moves focus to the previous focusable element |
