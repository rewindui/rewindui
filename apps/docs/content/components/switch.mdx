# Import
---
Import the `Switch` component using the following import statement.

```tsx
import { Switch } from '@rewind-ui/core';
```

# Colors
---
The available `color` values are: `blue`, `red`, `green`, `yellow`, `purple`, `gray`, `dark` and `black`.

# States
---
`Switch` component supports `disabled` and `error` states. The `error` state is enabled automatically by passing an error message to the `error` prop.

<View>
  <Switch label="Disabled" disabled />
  <Switch label="Click me!" error="Something went wrong..." />
</View>

```tsx
<View>
  <Switch label="Disabled" disabled />
  <Switch label="Click me!" error="Something went wrong..." />
</View>
```

# Description
---
The `description` prop can be used to display a description below the checkbox.

<View>
  <Switch label="Click me!" description="This is a description" />
</View>

```tsx
<View>
  <Switch label="Click me!" description="This is a description" />
</View>
```

Furthermore, a `ReactNode` can be passed to the `description` prop to display a more complex description that may contain multiple jsx elements.

<View>
  <Switch label="Click me!" description={<span className="text-purple-500 bg-purple-50 p-1 rounded">This is a purple description</span>} />
</View>

```tsx
<View>
  <Switch
    label="Click me!"
    description={<span className="text-purple-500 bg-purple-50 p-1 rounded">This is a purple description</span>}
  />
</View>
```

Similar styling can be achieved by using the `descriptionClassName` prop.

<View>
  <Switch label="Click me!" descriptionClassName="text-pink-500 bg-pink-50 p-1 rounded" description="This is a pink description" />
</View>

```tsx
<View>
  <Switch
    label="Click me!"
    descriptionClassName="text-pink-500 bg-pink-50 p-1 rounded"
    description="This is a pink description"
  />
</View>
```

# Ring
---
You can add a ring to the `Switch` by setting the `withRing` prop to `true`.

<View>
  <Switch label="Click me!" withRing={true} />
  <Switch label="Click me!" withRing={false} />
</View>

```tsx
<View>
  <Switch label="Click me!" withRing={true} />
  <Switch label="Click me!" withRing={false} />
</View>
```

# API Reference
---
## Properties

| Prop                 | Type                    | Description                                      | Default     |
|:---------------------|:------------------------|:-------------------------------------------------|:------------|
| color                | `SwitchColor`           | Sets the color                                   | `purple`    |
| disabled             | `boolean`               | Disables the checkbox                            | `false`     |
| description          | `string` or `ReactNode` | Description content                              | `undefined` |
| descriptionClassName | `string`                | Description span element classes                 | `undefined` |
| error                | `string` or `ReactNode` | Error content                                    | `undefined` |
| errorClassName       | `string`                | Error span element classes                       | `undefined` |
| label                | `string` or `ReactNode` | Label content                                    | `undefined` |
| labelClassName       | `string`                | Label element classes                            | `undefined` |
| radius               | `SwitchRadius`          | Sets the border radius                           | `full`      |
| size                 | `SwitchSize`            | Sets the checkbox size                           | `md`        |
| withRing             | `boolean`               | Shows a ring around the checkbox on active state | `true`      |

## Events

| Event    | Type                       | Description                              |
|:---------|:---------------------------|:-----------------------------------------|
| onChange | `(value: boolean) => void` | Fires when the switch state is changed   |

# Accessibility
---
The `Switch` component adheres to the [WAI-ARIA Switch Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/switch/).
